# PawKit
PawKit is a tool for hand gesture management in VRChat avatars.  
Breaking changes may occur until the project becomes stable.  
Please use it at your own risk.

## Required Packages
 - [Modular Avatar](https://github.com/bdunderscore/modular-avatar)
 - [Non-Destructive Modular Framework](https://github.com/bdunderscore/ndmf)
 - [Animator As Code V1](https://github.com/hai-vr/av3-animator-as-code)
 - [Animator As Code V1 - VRChat](https://github.com/hai-vr/animator-as-code-vrchat)
 - [Animator As Code V1 - Modular Avatar functions](https://github.com/hai-vr/modular-avatar-as-code)
